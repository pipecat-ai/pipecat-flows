- 0-argument and 1-argument (legacy) function handlers are deprecated and will be removed in 2.0.0. Update handlers to accept `(args: FlowArgs, flow_manager: FlowManager)`. A `DeprecationWarning` is emitted once per `FlowManager` instance the first time each deprecated shape is invoked. The `ZeroArgFunctionHandler` type is now exported so the currently supported handler shapes are fully described by the public type surface.
